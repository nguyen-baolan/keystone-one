# Project Overview: Keystone One

## Project Identity

- **Name**: Keystone One
- **Type**: Static Website (Hugo-based)
- **Repository**: https://github.com/nguyen-baolan/keystone-one.git

## Description

Keystone One is a Hugo-based static website built on the Hugoplate template. It's a modern, fast, and customizable website featuring a clean design with Tailwind CSS v4.0 integration.

## Core Features

### Content Features
- **Multi-Author Support**: Multiple content authors with profile pages
- **Blog System**: Full-featured blogging with posts, categories, and tags
- **Search Functionality**: Built-in search across content
- **Responsive Design**: Mobile-first, fully responsive layout

### Technical Features
- **Dark Mode**: User-toggleable dark/light theme
- **Hugo Extended**: Leverages Hugo's extended features for processing
- **Tailwind CSS v4.0**: Modern utility-first CSS framework
- **Hugo Modules**: Modular architecture with plugin system
- **Optimized Performance**: 95+ PageSpeed score target
- **SEO Ready**: Meta tags, sitemaps, and SEO optimizations

### Integration Ready
- Netlify deployment configuration
- Vercel deployment support
- GitHub Actions CI/CD
- GitLab CI support
- AWS Amplify configuration
- Disqus comments integration
- Google Analytics support
- Google Tag Manager
