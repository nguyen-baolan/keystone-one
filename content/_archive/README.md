# Content Archive

This directory contains archived content from the SaaS-focused version of the website before the institutional transformation on 2026-03-02.

## Purpose
- Preserve all original content for future reference
- Allow restoration if needed
- Maintain history of website evolution

## Structure
- `old-homepage/` - Original homepage content with SaaS messaging
- `old-sections/` - Section partials (call-to-action, featured-work, testimonial)
- `old-layouts/` - Original layout templates

## Status
All content in this directory has `draft: true` or is in a directory prefixed with `_`, making it invisible to Hugo builds and public visitors.

## Restoration
To restore any archived content:
1. Copy the content to its original active location
2. Remove `draft: true` from front matter if present
3. Update navigation/menus as needed
4. Test the site build

## Archive Date
2026-03-02

## Reason for Archive
Transformation from SaaS-focused messaging to institutional-grade positioning targeting mid-market enterprises ($10M+ revenue) and investment funds ($100M+ AUM).
